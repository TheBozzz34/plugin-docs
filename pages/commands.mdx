# All Commands:



- `/dsc`
- `/feed`, `/heal`
- `/day`, `/night`
- `/gmc`, `/gma`, `/gms`, `/gmsp`
- `/joke`
- `/getinfo`
- `/title`
- `/ban`, `/unban`
- `/mute`, `/unmute`
- `/reload`

## Usage:
Only some commands require arguments, `[this is required]`, `(this is optional)`

- `/mute <player> (reason)`   
- `/unmute <player>`
- `/ban <player> (reason)`
- `/unban <player`
- `/getinfo <player>`
