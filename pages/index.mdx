# Introduction
 
Refractor is a utility plugin I am currently developing. It started out as a project to learn Java and eventually became a collection of utilties I wanted to use on my server without the bloat of big plugins like Essentials
## Why use Refractor?

Easy, Refractor is **simple** and **powerful**. It has a variety of features intented to make server administration easier and quicker.
